# Code of Conduct

## Our Standards

- Be respectful and constructive in all interactions
- Focus on what is best for the community and the project
- Accept constructive criticism gracefully
- No harassment, trolling, or personal attacks

## Enforcement

Instances of unacceptable behavior may be reported to **conduct@dokrypt.dev**. All complaints will be reviewed and investigated.

## Scope

This Code of Conduct applies within all project spaces — issues, pull requests, discussions, and any other communication channels.
